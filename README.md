# 🎵 Music Player

A modern and interactive music player built with TypeScript, focused on clean architecture, dynamic audio interaction, and polished user experience.

The project includes playlist management, shuffle and repeat controls, animated sound visualization, dynamic background effects based on album artwork, and responsive UI interactions inspired by modern streaming platforms.

## 📷 Preview

![Music Player Preview](./preview/demo.gif)


## ✨ Features

- Play / Pause controls
- Previous and Next track navigation
- Shuffle and Repeat modes
- Interactive progress bar
- Volume control
- Dynamic playlist rendering
- Real-time sound wave animation
- Dynamic background effects
- Smooth transitions and animations
- Responsive interface

## 🛠 Technologies

- HTML5
- CSS3
- TypeScript

## 🎯 Purpose

This project was created to improve frontend development skills through:

- Component-based organization
- State and playlist management
- DOM manipulation with TypeScript
- Interactive UI development
- Responsive design principles
- Audio interaction in the browser
